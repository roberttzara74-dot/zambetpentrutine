# Pentru Tine 🌸

Un website premium, cinematic și interactiv, creat ca o mică poveste digitală — o experiență menită să aducă un zâmbet, construită integral cu HTML, CSS și JavaScript vanilla (fără backend, fără build step).

## ✨ Ce conține proiectul

- **Intro cinematic** — ecran de deschidere cu particule și petale animate (Three.js), text care apare progresiv și un buton de intrare cu tranziție de tip „zoom într-un univers roz".
- **Secțiune Hero** — mesaj de bun venit cu efect de scriere (typewriter), petale căzătoare animate cu GSAP.
- **Gânduri / mesaje** — carduri glassmorphism cu mesaje calde, animate la scroll.
- **Surprize ascunse** — elemente interactive (căpșuni, stele, funde) care dezvăluie mesaje surpriză la atingere, însoțite de mici explozii de inimioare.
- **Galerie polaroid** — cărți foto stilizate care se întorc la atingere, dezvăluind un mesaj pe verso.
- **Mini-joc** — găsește cele 5 căpșuni ascunse; la finalizare apare o animație de confetti + inimioare, desenată pe canvas 2D.
- **Final spectaculos** — cer înstelat animat (canvas 2D), text final care apare secvențial.
- Fundal ambiental animat cu particule (Three.js), efecte glassmorphism, blur, umbre moi.
- Complet **responsive** — funcționează pe telefon, tabletă și desktop.
- Respectă `prefers-reduced-motion` pentru accesibilitate.
